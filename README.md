🟡 Pac-Man – Консольная версия на 7 языках
Классическая аркада «Pac-Man» в упрощённом виде прямо в вашем терминале!
Управляйте Pac-Man’ом, собирайте точки, избегайте призраков и устанавливайте рекорды.
Реализована на семи языках программирования с единым геймплеем, цветной графикой, системой очков и рекордов, уровнями и бонусами.

🚀 Возможности
Игровой процесс – перемещайте Pac-Man по лабиринту, собирая обычные и супер-точки.

Призраки – четыре разноцветных призрака с простым ИИ (случайное движение или преследование).

Бонусы – супер-точки позволяют съедать призраков на время.

Уровни – после сбора всех точек открывается следующий уровень с увеличенной скоростью.

Счёт и рекорд – текущий счёт и лучший результат сохраняются в локальный файл.

Цветная графика – Pac-Man, призраки и элементы лабиринта окрашены.

Управление – стрелки для движения, Q для выхода, R для перезапуска.

Кроссплатформенность – работает в любом терминале с поддержкой ANSI (Linux, macOS, Windows с WSL или ConEmu).

📖 Использование
Синтаксис (единый для всех версий):

bash
<команда> [опции]
Опции
Опция	Описание
-s, --speed <N>	Базовая скорость игры (задержка в мс, по умолчанию 150)
-h, --help	Справка
Управление
Клавиша	Действие
← / A	Движение влево
→ / D	Движение вправо
↑ / W	Движение вверх
↓ / S	Движение вниз
R	Перезапуск после окончания игры
Q	Выход из игры
🛠 Установка и запуск
Python
bash
pip install windows-curses  # только для Windows
python pacman.py [опции]
C++
bash
g++ -std=c++17 pacman.cpp -o pacman -lncurses
./pacman [опции]
Требуется библиотека ncurses (установите через пакетный менеджер).

Go
bash
go get github.com/nsf/termbox-go
go build pacman.go
./pacman [опции]
JavaScript (Node.js)
bash
npm install blessed
node pacman.js [опции]
C#
bash
csc pacman.cs
mono pacman.exe [опции]   # или dotnet run
Требуется поддержка ANSI в терминале (на Windows используйте Console.OutputEncoding = Encoding.UTF8).

Ruby
bash
ruby pacman.rb [опции]
Java
bash
javac -cp .:lanterna.jar pacman.java
java -cp .:lanterna.jar pacman [опции]
Требуется библиотека lanterna (скачайте JAR и добавьте в classpath).

🧠 Логика игры
Лабиринт – фиксированная карта 20×20 со стенами, точками и супер-точками.

Pac-Man – управляется игроком, собирает точки (+10 очков) и супер-точки (+50 очков, временная возможность есть призраков).

Призраки – четыре призрака с разными цветами; движутся случайно, но при включении режима «испуг» убегают от Pac-Man.

Столкновения – если Pac-Man касается призрака в обычном режиме, игра заканчивается; если в режиме «испуг» – призрак исчезает и возвращается в центр.

Уровни – после сбора всех точек появляются новые точки и увеличивается скорость.

Счёт и рекорд – сохраняются в файл ~/.pacman_record.json.

✨ Дополнительные фичи
Анимированный Pac-Man – меняет направление «рта».

Режим испуга – призраки становятся синими и убегают.

Звуковые эффекты – при сборе точек и поедании призраков (ASCII bell).

Автосохранение – рекорд обновляется автоматически.
